# 🪙 نظام النقاط للمحاسبة

واجهة ويب بسيطة وفعالة لحساب وتحويل النقاط إلى أسعار وتسجيل العمليات المحاسبية.

## 💡 الميزات:

- 💾 **حفظ تلقائي** للبيانات في متصفح كروم باستخدام localStorage
- 🧹 **زر لمسح البيانات المحفوظة** بسهولة
- 💰 إدارة رصيد القاصة
- 🎯 واجهة RTL (من اليمين لليسار) مناسبة للغة العربية

## 🚀 الاستخدام:

1. افتح `index.html` في أي متصفح حديث (Chrome يوصى به).
2. أدخل بيانات المحاسب والمستخدم والنقاط.
3. سيتم حفظ البيانات تلقائيًا حتى بعد الإغلاق.
4. استخدم زر "🧹 مسح البيانات المحفوظة" لإعادة تعيين كل شيء.

## 🛠️ التقنيات المستخدمة:

- HTML5
- CSS3 (بتصميم داكن)
- JavaScript (localStorage)

## 📁 ملفات المشروع:

- `index.html` — الواجهة الرئيسية مع جميع الوظائف.

## 📜 الترخيص:

هذا المشروع مفتوح المصدر ويمكنك استخدامه وتعديله بحرية.
